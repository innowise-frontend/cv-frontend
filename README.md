# cv-frontend

1.Clone this repo:

```sh
git clone https://github.com/innowise-frontend/cv-frontend.git
```

2. Install `pnpm`:

```sh
npm install -g pnpm
```

3.Set the GraphQL URL in `.env`:

```sh
VITE_GRAPHQL_URL="http://localhost:3001/api/graphql"
```

4. Install dependencies:

```sh
pnpm i
````

5. Run app:

```sh
pnpm run dev
```
